# nnn-select

Select a file in [nnn](https://github.com/jarun/nnn/) from another program!
```
nnn-select [path]
```

## Build and Install

Build dependencies:
 * golang
 * make

Optionally configure `config.mk` to specify a different install location.
Defaults to `/usr/local/`

`sudo make install`

## Uninstall

`sudo make uninstall`
